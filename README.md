# codex-ux-skills

Neutral, reusable Codex skills for UX writing, form UX auditing, and WCAG 2.1 AA web UI audits.

## Included skills

- `email-ux-best-practices`: review and improve product email copy and structure
- `form-ux-best-practices`: audit and rewrite form UX, validation, and accessibility
- `wcag-21-aa-web-ui-audit`: run accessibility audits and generate remediation outputs

## Repository goals

- Keep skills product-agnostic and portable across projects
- Maintain standalone skill docs, templates, and helper scripts
- Avoid coupling to app-specific branding, paths, or runtime assumptions

## Usage

Install individual skill folders using your Codex skill installer workflow, or copy a folder directly into your local skills directory.
